通过刷二维码连接wifi

从GOOGLE PLAY上下载QR Code Generator，选择Wi-Fi，输入如下信息：

Network Type :WPA/WPA2
Network Name(SSID):Q... #wifi名字
Password:h... #wifi密码
点击`Generate`就生成了一个二维码，家里来了客人，通过手机刷这个二维码，就可以连接到wifi，不需要口头告知。 其实这个二维码是一条字符串： 
WIFI:WPA;S:Q...;P:h...;;
