GNU Emacs

Posted on 2018-02-15 | In Coding , Network , 未分类

官网：GNU Emacs

一、安装

从官网下载Windows版本，例如：emacs-25.3_1-x86_64.zip（问题：sig扩展名的是啥?），解压到自己的目录后，用bin/runemacs.exe启动Emacs。

从share/emacs/readme.w32中可以查询到如何安装配置Emacs。参考windows7 安装Emacs 24

运行bin/addpm.exe，就可以从Windows开始菜单Gnu Emacs启动Emacs了。第一次安装打开Emacs，请认真学习欢迎界面的Emacs快速指南：一些基本的移动命令

打开用户目录C:\用户\用户名\AppData\Roaming，看到已经创建.emacs.d目录了

下载http://download.savannah.gnu.org/releases/color-theme/的color-theme-6.6.0.zip，解压到上述目录 #翻墙！

创建.emacs文件，重新启动Emacs，就变成黑色背景了。 #为啥没有生效，还是白色的！？
.emacs文件内容如下：
(let ((default-directory “~/.emacs.d/color-theme-6.6.0/“))

(normal-top-level-add-to-load-path ‘(“.”))

(normal-top-level-add-subdirs-to-load-path))

(require ‘color-theme)

(color-theme-initialize)

(color-theme-arjen)
二、入门

光标移动：

翻页：

M-v：page up C-v：page down C-l：切换鼠标屏幕位置

箭头：

C-p：上 C-n：下 C-b：左 C-f：右

首尾：

M-b：左 M-f：右 C-a：行首 C-e：行末 M-a：句首 M-e：句末 M-&lt;：文首 M-&gt;：文末
窗口：
C-x 1：关闭其他窗口，注意是1（one），不是l（el）
