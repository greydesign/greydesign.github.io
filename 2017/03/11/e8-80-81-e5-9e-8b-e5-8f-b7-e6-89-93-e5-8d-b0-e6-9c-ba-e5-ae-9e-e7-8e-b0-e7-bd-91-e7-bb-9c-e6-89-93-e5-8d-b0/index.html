HP Laserjet 1008 Windows 10驱动问题以及老型号打印机实现网络打印

Posted on 2017-03-11 | In IoT , Network , VPS

Windows 10 环境下，打印机内置的驱动程序无法安装，从hp官网下载的那个才4M的驱动也不好用！
下载win10下的完整驱动，安装Laserjet 1100的驱动就可以了。
参考：Windows10全新安装后为什么惠普打印机P1108的驱动无法安装

亟需解决的问题：如何在linux下查看打印机设备的状态？

最好路由器、树莓派不需要针对打印机类型安装驱动程序，在客户端设置，因为我有照片打印机，也有黑白激光打印机。

二、通过路由器实现网络打印

主题：求推荐一个能把打印机变成网络打印机的路由器
首先，路由器要改刷openwrt固件，这个步骤，联线newifi比小米方便些，改刷后，两个路由器功能是一样的（硬件也基本一样）。
然后安装并启动luci-app-p910nd，路由器就改造完成，将打印机接到路由器USB口。
打印机安装，以windows10系统为例，新建本地打印机，新建tcpip端口（填上面改造好的路由器IP地址），然后安装打印机驱动（原装打印机驱动）即可。

三、通过树莓派实现网络打印

sudo apt-get update
sudo apt-get install hplib #安装惠普打印机驱动
sudo apt-get install cups #安装CUPS
sudo usermod -a -G lpadmin pi #用户pi添加到local-preference管理员组里，以使用户pi有权限对CUPS进行配置
sudo nano /etc/cups/cupsd.conf #修改CUPS配置文件，找到Listen localhost:631，改localhost为0.0.0.0，在三个&lt; /Location &gt;前面加上Allow from @LOCAL，让网络用户拥有管理权限 sudo service cups restart #重启CUPS

在浏览器中输入http://your-raspi-pi:631/就可以对树莓派上的CUPS服务进行访问。
在设置界面添加打印机：Administration，Add Printer，就会看到问的HP LaserJet Professional P1108 USB 000000000Q838XR4SI1c HPLIP (HP LaserJet Professional P1108)打印机，注意，一定要选择那个带有USB标识的！还有，我怎么知道这个usb是台式机的usb还是树莓派的usb？下一步注意选择Share this Printer添加后，打印机地址为：
https://192.168.1.232:631/printers/HP_LaserJet_Professional_P1108-Pi。注意，为了区别于台式机连接，我在打印机名称增加了-Pi字符。
打印测试页，失败，提示为stopped "Filter failed"，为啥！！！？
注意，在windows系统中添加网络打印机，用http访问，不要用https。
在手机端，通过Google Play安装HP Print Svevice插件，已安装“HP Inc.”服务，点击即可启用。
问题：魅族的打印菜单在哪里，似乎是个隐藏的菜单，而且提示没启用。

问题：我在三个Location都添加了Allow from www.greydesign.tk，难道也无法远程访问实现打印？会返回错误：

Bad Request #用http://www.greydesign.tk:631访问失败

四、谷歌云打印

打开chrome浏览器，确保登录了自己的google账号，在设置中，找到显示高级设置，管理谷歌云打印，就看到局域网中的打印机，在树莓派上的惠普打印机也在列表中，都添加上。

五、PrinterShare

通过google play安装了PrinterShare应用，选择打印机，附近WiFi，能看到连接在树莓派上的HP LaserJet Professional P1108，但是连接会提示Connnection refused错误，可能和用了https有关。但是选择Google Cloud打印机，则没有警告，打印测试页没反应，估计是用了原来安装在台式机上的打印机，没有正确识别出树莓派上的打印机。
