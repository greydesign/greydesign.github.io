周总结 #没有总结语文测验成绩。

数学：一个待达标和两个良。
语文：一个待达标两个良和一个优。
英语：没有。
总体来说，这一个周的情况非常不好，居然右两个待达标和一个优和四个良！下周一定要加倍努力。
和手机不得不说的故事 #被爸爸发现周五回家一直玩手机到5点18分，还忘了6点准时练琴。 
星期五：4点15分左右上了微信，到了差不多五点五分的时候就不动手机了。
星期四：回来一段时间后看了一会微信，然后忘关流量了。大家一直在说话，所以一直在消耗着。
星期三：在微信上和同学说话，具体时间忘了。还看了公众号上的文章。
星期三以前的不记得了。
语文第六课测60分 #此次测验，全班95以上13人，达标4人，待达标1人。 
 在这周星期二的时候，我们四班学了语文书的第六课：《卢沟桥烽火》。接下来就考了一个课后测验题，同时也测试一下我们的现场学习能力，看看我们在这一次上课的时间里，到底有没有学会本课全部讲过的、本课的重点之类。
而我是一个刚刚学完的时候，基本上是几乎都不会的。所以我这次测验题看着的时候就感觉不怎么会。做的时候看着也有一点点晕晕的，感觉根本想不起来到底是什么答法，到底应该怎么样才能写出这道题的解答方法。然后，就因此我才考了60分。
