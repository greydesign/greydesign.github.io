四、如何加速树莓派的外网响应速度

树莓派访问非常慢，比自家的路由器管理界面慢多了，为啥？是否需要买个最新的树莓派3，能外接ssd卡吗？
解决树莓派磁盘速度问题 | 树莓派的 Docker 之旅

参考 树莓派 之 提升SD卡读写，实际操作：

sudo apt-get install hdparm #安装hdparm

hdparm -Tt /dev/mmcblk0 #提示错误：/dev/mmcblk0:Permission denied，咋回事？
待补充：树莓派的文件系统介绍

待实施的操作：
如果你的存储卡是 Class 10，可以启用 sd overclock 来提升读写：
echo “dtparam=sd_overclock=100” | sudo tee –append /boot/config.txt
重启后生效

五、树莓派安装usb无线网卡

第一代树莓派没有内置wifi与蓝牙，花钱买raspberry pi 3会有负罪感，拿出尘封的Mercury网卡吧。参考如下文章：

树莓派无线USB网卡，Mercury WIFI配置

由于系统很新，lsmod、lsusb、ipconfig -a都能确认驱动已安装好了，wpasupplicant系统不知何时已经是最新的了。修改配置文件interfaces，将wlan0 的manual改为dhcp，就顺利连接上家里的无线路由器。

需要注意的几点：
1、我的wifi加密算法是AES，pairwise和group项都是“CCMP”;
2、重启网络，也可以用单独启动wlan0代替：sudo ifup wla<span class="hljs-symbol">n0</span>
